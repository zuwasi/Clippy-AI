# Contributing to Clippy AI Assistant 📎

We welcome contributions! The easiest way to contribute is by adding new **90s nostalgia slogans**.

## Adding New Slogans

1. Fork the repo
2. Open `index.html`
3. Find the `nostalgicSlogans` array (~line 445)
4. Add your slogan string to the array:
   ```js
   "Your amazing 90s wisdom here. 📎",
   ```
5. Submit a Pull Request with the title: `slogan: <short description>`

### Slogan Guidelines

- Keep the 90s nostalgia vibe — Windows 95/98, dial-up, floppy disks, Bill Gates, retro tech
- Use emojis for personality 📎💾🎵
- Be funny, cheeky, and over-the-top nostalgic
- Keep it under ~120 characters for best display in Clippy's speech bubble
- No offensive or inappropriate content

### Example Slogans

```
"The 90s had no cloud storage. We had SHOEBOXES FULL OF FLOPPIES. And it was glorious. 💾"
"Bill Gates didn't need a smartphone. He had VISION. 👓📎"
```

## Other Contributions

- **Bug fixes** — always welcome
- **New features** — open an issue first to discuss
- **Translations** — add new language entries in the `UI_STRINGS` object
